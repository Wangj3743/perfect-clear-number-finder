What PC Am I On?

This is a perfect clear (PC) number finder for Tetris.


On an empty board, How many pieces have been used?


use this input by typing the actual piece count (should be a multiple of 5)
e.x. 125


or use this input for multiples of 10 and remove the last digit (which is a 0)
e.x. 45 (450 -> "45")


or use this input for multiples of 5 and remove the last digit (which is a 5)
e.x. 23 (235 -> "23")
